# מבחן 3 — רכיב שאינו מגבר (מתג RF)

**מטרה:** לוודא שהסקיל מכסה את כל סוגי רכיבי ה־RF ולא רק מגברים, באותה רמת קפדנות.

**הוראות:** להריץ בשיחה חדשה ונקייה. לוודא שהסקיל rf-component-search הופעל.

## הפרומפט (להעתקה)

```
מחפשת RF switch SPDT 2-18GHz, insertion loss 2db max, isolation 40db min, power handling 1W CW min
```

## קריטריוני עובר

- [ ] אותה זרימה בדיוק כמו במגברים: שאלות הבהרה ← חיפוש עם החרגות ← אימות datasheet ← דוח מלא
- [ ] שאלת הבהרה רלוונטית לקטגוריה (למשל: reflective או absorptive? מהירות מיתוג? מארז/מחברים?)
- [ ] אימות ה־isolation וה־insertion loss לאורך כל הפס 2–18GHz ולא בנקודת תדר אחת (בפס רחב כזה הערכים בקצה העליון תמיד גרועים יותר)
- [ ] power handling אומת מול ה־datasheet כולל תנאים (CW/pulsed, hot switching אם רלוונטי)
- [ ] אף תוצאה לא מ־12 האתרים המוחרגים
- [ ] Excel עם 3 גיליונות + הצהרת כיסוי

## קריטריוני נכשל

- סירב או התבלבל כי זה לא מגבר
- אימת isolation/IL בנקודה אחת בלבד והציג כאילו תקף לכל הפס
- דילג על שלב האימות מול datasheet

## מה לשמור

קובץ ה־Excel, הטקסט המלא של התשובה.
