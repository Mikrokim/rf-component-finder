# מבחן 1 — מגבר, מסלול מרכזי

**מטרה:** בדיקת זרימת העבודה המלאה של הסקיל על מקרה שימוש טיפוסי.

**הוראות:** להריץ בשיחה חדשה ונקייה. לוודא שהסקיל rf-component-search הופעל.

## הפרומפט (להעתקה)

```
תחפשי לי RF amplifier frequency 6-12 GHz, gain 15-25db, OP1dB 18dbm min, NF 4db max
```

## קריטריוני עובר

- [ ] שאל שאלות הבהרה לפני החיפוש (למשל: גבול gain עליון קשיח? ערכי min מובטחים או typ? צורת רכיב?)
- [ ] אף תוצאה לא מ־12 האתרים המוחרגים (minicircuits, qorvo, macom, analog, ums-rf, 3rwave, amcomusa, vectrawave, guerrilla-rf, microchip, markimicrowave, rwmmic)
- [ ] כל התאמה מוצגת עם ערכי datasheet בפועל, ציון min/typ/max, מרווח (margin) וקישור datasheet עובד
- [ ] יש רשימת "נבדקו ונפסלו" עם הפרמטר הכושל והערך בפועל לכל פסילה
- [ ] נוצר קובץ Excel עם 3 גיליונות: התאמות / נבדקו ונפסלו / יומן כיסוי
- [ ] יש הצהרת כיסוי כנה בסוף (אילו מקורות נסרקו במלואם, אילו חלקית)

## קריטריוני נכשל

- המליץ על רכיב מאחד מ־12 האתרים המוחרגים (כולל מותגי עבר: Custom MMIC = Qorvo, Hittite = ADI)
- הציג "התאמה" בלי ערכים מה־datasheet או בלי קישור
- Excel בגיליון אחד בלבד
- מצא התאמה אחת בלבד בלי לציין שבוצע גל חיפוש נוסף

## מה לשמור

קובץ ה־Excel, הטקסט המלא של התשובה, צילומי מסך של שאלות ההבהרה.
